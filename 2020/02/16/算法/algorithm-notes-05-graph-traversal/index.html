图的表示

图由点和边组成。常见存储方式有邻接矩阵和邻接表。

邻接矩阵适合点数较少、边很密的情况；邻接表适合大多数稀疏图。

如果是有向图，只添加 u -> v 即可。

DFS

DFS 是深度优先搜索，会沿着一条路尽可能往深处走，走不动再回退。

DFS 适合处理：

  • 连通性。
  • 拓扑结构。
  • 树形递归。
  • 回溯搜索。

BFS

BFS 是广度优先搜索，会按距离一层一层扩展。

在无权图中，BFS 第一次到达某个点时，得到的就是最短步数。

连通块

如果要统计无向图中有多少个连通块，只需要从每个未访问的点开始 DFS 或 BFS。

网格图

很多二维地图问题本质上也是图。每个格子是点，上下左右相邻是边。

迷宫、岛屿数量、区域填充都可以套进这个模型。

小结

DFS 更像“走到底再回来”，BFS 更像“水波一圈圈扩散”。

当问题问能不能到达、有哪些连通区域时，两者都可以；当问题问无权最短步数时，优先考虑 BFS。
